# File check
> Check if a path is a file

## Installation
```bash
npm install --save file-check
```

## Usage
```javascript
const check = require('file-check');

if (check('my-file')) {
  console.log('is a file');
}
```

> Working example in [test.js](https://github.com/tobihrbr/dir-check/blob/master/test.js)
